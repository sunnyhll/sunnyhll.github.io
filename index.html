Window Message

发表于 2024-01-31

窗口消息

GUI应用程序必须响应来自用户和操作系统的事件。

来自用户的事件包括用户与程序交互的所有方式：鼠标点击、按键、触摸屏手势等等。
来自操作系统的事件包括程序之外的任何可能影响程序行为的东西。例如，用户可能插入一个新的硬件设备，或者Windows可能进入低功耗状态(睡眠或休眠)。
这些事件可以在程序运行时的任何时间发生，几乎可以按任何顺序发生。如何构造一个不能预先预测执行流（flow）的程序?

为了解决这个问题，Windows使用了消息传递模型。操作系统通过传递消息与应用程序窗口进行通信。消息只是指定特定事件的数字代码。例如，如果用户按下鼠标左键，窗口将接收到具有以下消息代码的消息。

C++

#define WM_LBUTTONDOWN 0x0201

一些消息具有与它们相关联的数据。例如，WM_LBUTTONDOWN消息包含鼠标光标的x坐标和y坐标。

要将消息传递给窗口，操作系统将调用为该窗口注册的窗口过程。(现在你知道窗口程序的作用了。)

https://learn.microsoft.com/zh-cn/windows/win32/learnwin32/your-first-windows-program

窗口句柄

窗口句柄是消息传递的源和目的，在每一个进程空间窗口句柄都是局部的，需要通过窗口名字去发现，对于某个窗口消息队列的监测，可以通过visual studio自带的spy++去定位监听并且调试。

个人感悟

《领导力》读后感

成为领导者的基本能力

领导者是需要一种成就他人，成就团队的能力，相较于个人自身能力，反而不是最重要的。本书的作者是一名职业足球教练，从职业足球选手退役后，变成了一名带队教练，积累了多年的经验并且成功成就了一支超强的足球队伍，总结他的教练生涯上成功的关键因素。

  • 聆听

    • 聆听是最直接的了解对方和自己的方式，成功者的经验和话语或多或少反应了他的价值观，也对自己会有一定的启发，没有利益冲突人的忠告往往是真诚和有用的，要自己思考和体味其中价值并未自己带来改变。

  • 观察

    • 细节观察：对于有重要影响力的细节，需要着重观察，但是不能深陷细节的泥淖，丢失全局的目标。

    • 整体观察：局势，核心要素，评估对手，是整体观察的重要因素，5W准则，回答Why，What，Where，Who，When问题。

  • 阅读

    • 对于你关注的领域，永远需要孜孜不倦的热忱去提升你对领域的认识，外界环境可能会对你的汇报不是同侪对等的，但正确的努力方向，一定会带给你想要的汇报。万物的哲学规律是相同的，不要局限于同意领域的经验，有时候新的领域的成功经验也会带给你新的思路。

领导者青睐的对象

领导者和成功的团队之间是有相互吸引的作用力的，即使会有小的嫌隙，但总的方向上，一定是默契的。

  • 纪律

    • 纪律性是一支队伍需要有有种一致的信念，对于不遵守纪律的队友，在权力范围内，可以更换掉，当然在惩罚力度上需要合适，避免过于严苛打消队伍的积极性。足球俱乐部的惩罚力度分为，罚工资，坐冷板凳，停赛和转会，职场上也是类似，言语警告、罚奖金、减少项目机会，没有新项目和转组换公司。惩罚的后果除了提振队伍纪律，也会让队伍丧失一些机会，也是团队领导者需要承受的损失。

  • 工作效率

    • 不想当将军的士兵不是好士兵，求胜欲和上进心是一个队员成功的必须条件，努力努力再努力是内心的渴望的一种体现，无论是童年的苦难还是内心的渴求驱使，一个人永不言弃的追求是非常宝贵的。这种渴求会让队员拥有超乎寻常的刻苦，成就超人的效率，不是因为拥有单位时间更加高的天赋，而是更多的时间精力投入收获了更多的效果。

  • 能动性

    • 能动性是一个人意志力的表现。永不服输，渴望成功的心情是每个人都有的，但是意志力随着环境、时间推移，不变的人是很少的，超凡的意志力能够让一个人持续进取，持续输出，构建并实现出宏伟的蓝图。

  • 信念

    • 足球是团体竞技体育的典型代表，个人力量是无法支配比赛的，因此球员的信心在漫长的90分钟比赛期间的起伏也会影响比赛的结局，哀兵必胜，但是萎缩的信心是永远无法带来胜利的。

领导者对团队的建设

足球比赛就像一场战役，队员用血肉之躯去竞技，当中会有心情浮动，体力消耗，环境因素，早就一支长盛不衰的队伍是需要有意图的训练，并且临场灵活的调度。

  • 组织：团队的组织架构和信仰是持续稳定的，相对来来走走、个性不同的球员，团队的组织架构能够带来良好的运营与稳定性。

  • 准备：团队赛前准备充分，就能灵活应对赛场上的灵活多变，战术上的充分准备能够让队员更加理性的看待战况，打出自己的水平。

  • 管道：球队的运营和公司很像，在流动的队伍中创建持续的质量输出，培养球员，高薪聘请，运动员的运动生涯周期，都会影响球队的成绩。类似公司的职级架构，球队的组织层次分为训练营、B队、A队、预备队和首发阵容。培养的重心也会从初级的技术体能转而向情感和心理需求管理。球员之间的情谊也是很重要，这对团队建设也是非常重要，有时候能够起到一加一大于二的效果。

构建吸引人的团队

  • 团队合作：团队平均年龄是管理者考量的一个因素，平衡的结构有助于队伍保持高速发展，不是说年纪大了就没有用，但是对于运动员而言，年纪大了不够核心被迫选择离开也是两难的境地。领导者不要因个人喜好去选择人才，而是需要根据团队需求去吸收希望有的人才，构建核心团队。

  • 队长：需要具备如下四个特质：

    • 有领导欲望：
    • 能够传达教练意愿：
    • 队伍中具有威信
    • 能够随机应变：

领导者如何评价团队队员的水平

  • 优秀的榜样：任何领域都会有榜样，世界级的榜样屈指可数。世界会有完备的评分体系，去评估每一位队员的个人能力，不难发现优秀人才。给团队指定合理的目标能最大化进展，太高或者太低目标都不能达到最大化效果。

  • 鼓舞：队员潜力开发需要鼓舞，不需要和队员打成一片，适当的敬畏是需要的，这是指挥者必须要遵守的原则。对于每个队员的激励因素是不一样的，而且不要随便变化！教练的动力就是获得冠军，队员都需要为这个目标去努力。对于有效的鼓励和监督场景，领导者不要缺席。

  • 自满：自满情绪会造成失败，无论对领导者还是队员，哀兵必胜。即使现在已经有成就，吸引人的永远是下一个冠军就。

领导对人才的态度

  • 求职面试：领导面试求职者时，希望发现发现面试者有何雄心、是否只是把工作当作跳板。除了素质和资历，充实程度也很重要。领导需要寻找的时热情、积极态度、眼神的交流和个人的勇气。

  • 人际关系网：在职场中，人际关系网非常重要，值得花时间去培养，别人的建议往往能够拨开迷雾。当然人机关系网需要有方向的建立，足球队教练的人际关系网，除了同侪，现役球员，还有前任球员，球队支持者。对于普通职场的人，除了领导，下属，团队之外，没有绝对利益相关的同事也可以成为人际关系网的一部分，甚至在公司外部，也需要关系网，人不是孤独的一座小岛，人生漫漫长路一定需要互相扶持帮助。

  • 炒鱿鱼：职业运动生涯的淘汰率时很高的，避免不了严格的淘汰机制，解雇的场面一定时尴尬和令人难受的。尽可能的帮助别人找到合适的领域，也是一份绵力。

领导的自我诉求

  • 时间：每个人每天都只有24小时，能人是如何具有超乎常人的产出？一定是超出常人的付出，更多的时间花费在自己想要深耕的领域，长久下来就会有变化。

  • 分心：专注在一个领域对于喜欢这个领域的人而言，不是大问题，把精力集中在重要的事情上，建立良好的个人特质，有利于长期的发展。近现代的生活充斥太多令人分心的东西，让人眼花缭乱，这些事情会让时间耗费在不留痕迹的投入中，并且得不到任何的收益。对于足球运动员而言，读书也是一种分心，当然对普罗大众而言，读书的必须的。

  • 失败：积极态度面对失败，每个人都会有失败不要害怕。人对于生存下去的渴望是有本能的，避免失败，就能成功。对于顶级足球教练而言，球队在赛季的胜率也就不过72%，也就说再优秀的人也面临过失败。在失败中从自身找问题，能够更好的利用失败成长。同时，失败也让我们知道了自己的弱点，和优势，并且同时知道如何调整自己的方向和展示角度。

  • 批评：批评对于批评者而言，是容易的，因为他不关心执行过程中的细节和微妙心态，只需要专注于找到不足之处。而对于执行者而言，面对批评的压力，正确的看待是有益处的。即使最坏的事情发生了，也不会是世界末日。当然再强硬的人也会有软弱的时候，这段时期，需要重新建立信心，和主动和自己和解。

领导力传递

  • 说：跟各种人打交道，不同的人需要采取不同的方式。养成换位思考的习惯，把自己想象成聆听者，关注技艺精湛的公共演讲者。

  • 写作：作者出书，我第一本书《北部之光》（A Light in the North）,第二本书是《管理我的生活》（Managing My Life），忙里偷闲时胡乱写下自己的想法和记忆的好习惯能够有助于写作。

  • 回答：与媒体打交道可以呈现出不同的形式，但是其中的关键就在于控制。情感和肢体语言不要受到环境的影响。

领导与管理的不同

我的工作是制定非常高的目标。我的工作是让人们相信自己能够做到之前认为自己做不到的事情。我的工作是画出一条之前从未走过的道路。我的工作是让每个人都相信世界上没有不可能。这就是领导和管理之间的区别。

  • 对老板汇报：对于那些你需要进行汇报的负责人，了解其性格是何等重要。

  • 控制力：无法控制航线、调整航速的船长是不能够安全抵达港口的。完全控制是有效领导的必要条件，但是细节和micro管理是两回事。

  • 授权：随着事业的发展，其团队成员也越来越多。这就要求对其周围人员进行授权。如今所有的大型俱乐部都有首席执行官，由其负责全部的商业活动并确保账目平衡，这对于很多俱乐部来说，不至于太过失控。因此我让大卫·吉尔负责电视合同、拉拢赞助商、敲定球员合同的细节、管理财政和市场营销部、应付稽查员和律师、确保俱乐部遵守健康与安全条例以及适用于所有集团的法律法规。领导者在乎细节，而具体工作需要他们来做。

  • 决策：有效的授权依靠的是其他人做出决定的能力。做出决定所依据的是你目前可以使用的信息，并且在正确的时机。

销售思维

领导者需要擅长销售自己的想法，说服下属或者对手方去做他们不想做的事情。作者作为足球教练，主要在如下几个专项上施展自己的领导力。

  • 买球员: 教练为了组建一支超强队伍，需要不断去搜寻和购买具有潜力的年轻队员，就像职场中，也需要可以培养良好的梯度，为了队伍的正常新老更新做准备。

  • 节约：总教练节约，可以为队伍日常开支进行一定的节约。

  • 报酬：球员的报酬，除了考虑个人能力，也要考虑同侪效应，整体蛋糕切分尽量公平为了整体稳定性。

  • 谈判：和下属谈判，是领导不可避免的问题，确立好目标和底线，在平稳情绪过程中进行，保证谈判结果的合理性。

  • 经纪人：经纪人作为领导者和潜力下属的中间人，也是不可避免的角色，了解经济活动作为中间人的特性，能够良好的避免其带来不良的影响。

技术与信息

技术在新时代为球队，球员的培养提供了重要的指标和导向。现代化的队伍建设量化了每个球员的特性，科学化培养成了标准流程，让领导对于每一个下属的培养有了更加标准化流程，不再需要发挥个人特性，熟悉流程可以合适培养大多数的人才。

  • 创新：技术革新和科技理论的更新创造了新的培养队伍方式，领导者需要紧跟时代进行灵活升级。

  • 数据过载：现在的传感器技术已经能够对每一个小事进行细节捕捉，然而这样也造成了数据太多的问题，不要完全淹没在数据海洋里，有时候人的经验和直觉往往不需要海量数据的挖掘，也会更高效。

  • 机密：队伍成功是必须要有保密守则，对于领导注重的信息，不能擅自泄露，对于违反规定的人，必需找机会以提醒甚至惩戒。

竞争与环境

  • 对手：对手的不理智行为，往往会影响决策者的正确性，同时也容易陷入博弈论的恶性循环中，需要理智并且妥善化解。

  • 全球市场：全球化的人才市场为雇佣关系带来了更加充分的市场竞争，有助于企业和队伍融合成长。

  • 新的节奏：到了新的生活阶段，即使一个成功的领导者也需要适应新的节奏，培养新的兴趣爱好来填满自己的生活。

《非暴力沟通》

非暴力沟通四要素

  • 首先，留意发生的事情，清楚地表达观察结果，而不判断或评估。
  • 接着，表达感受，例如受伤、害怕、喜悦、开心、气愤等等。
  • 然后，说出哪些需要导致那样的感受。
  • 最后，提出非暴力沟通的第四个要素——具体的请求。这一要素明确告知他人，我们期待他采取何种行动，来满足我们。

观察误区

  • 道德评判：对他人的评价实际上反映了我们的需要和价值观。应该直接说出我们的价值观，而不会指责他人。
  • 比较：比较也是一种评判。
  • 回避责任： 我们可以用负责任的语言代替回避责任的语言。补充我为什么要这么做，以及自己的感受。
  • 强人所难：我们无法强迫他人按我们的期待生活。认为“某人应当受到惩罚”使我们难以体会到心中的爱。

区分观察与评论

将观察和评论混为一谈，人们将倾向于听到批评，甚至会产生逆反心理。不带评论的观察是人类智力的最高形式。

体会和表达感受

  • 示弱有助于解决冲突。
  • 区分感受和自我评价：当我们说“我觉得”，我们常常并不是在表达感受，而是在表达想法

前缀和

前缀和思想是运用前缀和数组作为桥梁，取解决数组的各种子数组优化问题。已知数组A[ i]，

s[ i] = s[ i - 1] + A[ i]

字段和从l 到 r则表达为：

sum(l, r) = s[ r] - s[ l - 1]

当A中都是非负数时，前缀和数组s单调递增。这样子数组和问题就转化成了s数组元素的问题。

递推关系

s[i][j]= s[i-1][j] +s[i][j-1] - s[i-1]s[j-1] + A[i][j]

子矩阵和

以[p][q]为左上角元素，[i][j]为右下角元素的子矩阵的元素和。

递推关系

sum(p,q,i,j) = s[i][j] +s[i][q-1] - s[p-1]s[j] + s[p-1][q-1]

其中，p，q总是需要减一。
